# Complete the MPD maturity assessment

## Why

The shipped maturity document claims to contain scores but omits them. That makes the
requested assessment non-verifiable even though the underlying controls and evidence are
present.

## What Changes

- Add an explicit five-level maturity model and scored dimensions.
- Separate implemented controls from observed effectiveness and missing evidence.
- Record risks, priorities, and evidence-backed next steps without changing runtime policy.

## Capabilities

### New Capabilities

None. This is a correction to the existing durable assessment, not a new runtime or
policy capability.

### Modified Capabilities

None.

## Impact

Documentation and OpenSpec records only. No source, config, dependency, runtime behavior,
or release artifact changes.
